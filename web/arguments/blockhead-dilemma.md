inference form: constructive dilemma. The blockhead must be read either as a physical system or as a
merely conceivable one; each reading deprives argument-lookup-table-objection of its force.

1. Physical horn. If the blockhead is a physical lookup table, it cannot exist
   (claim-lookup-table-physically-unrealisable) — and a fortiori in Block's own setting, a table
   bounded to a *human lifetime* of inputs is larger still than the LLM case (more varied inputs,
   far longer sequence; see claim-lookup-table-size-exponential). So there is no actual system to
   point at, and the "no mind" reaction is pumped by an object that could never be instantiated.
2. Conceptual horn. If the blockhead is merely conceivable, the "no mind" verdict
   (claim-lookup-table-lacks-mind) is not compelled (claim-imaginary-blockhead-no-secure-verdict):
   imagining a cosmic abstract object underdetermines whether it is a subject, and conceivability is
   symmetric — one can equally imagine it conscious.
∴ On neither reading does the objection secure its conclusion: physically there is no case, and
   conceptually the case yields no forced verdict. The intuition pump behind premise (2) of
   argument-lookup-table-objection (claim-lookup-table-lacks-mind) is therefore disarmed.

bearing on the targets: `attacks` argument-lookup-table-objection (and its premise
claim-lookup-table-lacks-mind) by removing the intuition's warrant rather than by conceding behaviour
without mind. Note the asymmetry with the conceptual point about Block's broader thesis: a *possible*
blockhead might still make the conceptual claim "behaviour ≠ mind" if its no-mind status were secure —
but horn 2 is precisely that, for an impossible-and-only-imaginable object, that status is *not*
secure.

weakest premise: horn 2 risks proving too much — if intuitions about imaginary minds carry no weight,
the same deflation hits thought experiments the functionalist *likes* (e.g. silicon-isomorph
intuitions favouring claim-substrate-independence). A principled asymmetry is owed, and is
supplied by argument-intuition-domain-asymmetry: the cosmic table is a far extrapolation from any
paradigm mind (in scale and structure), whereas a brain-scale organisation-preserving isomorph
(concept-grain) is a near neighbour — so withdrawing warrant from the former intuition does not
withdraw it from the latter (claim-blockhead-deflation-targeted-not-global). Block's residual reply
relocates the verdict from *scale* to *kind* (brute storage is mindless at any size); answering it
requires denying that "pure storage" is a stable kind at mind-matching scale — the live crux recorded
there.

Distinct from argument-lookup-table-physical-impossibility: that establishes the physical horn and
attacks the application-to-actual-LLMs step; this is the *dilemma* that adds the conceptual horn to
deflate the objection even on Block's intended (in-principle) reading. Distinct from
argument-lookup-table-combinatorial-impossibility, which denies the table matches the behaviour at
all (a competence gap), not the legitimacy of the intuition.
