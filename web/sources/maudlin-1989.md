relevance (content): introduces the *Olympia* machine — a device that reproduces the actual sequence of
physical states of a conscious computation but lacks the counterfactual sensitivity computationalism
requires for genuine implementation — and argues that supplying that counterfactual structure with inert,
never-triggered apparatus changes no actual physical activity, so cannot change whether there is
experience. The source for [olympia](../arguments/olympia.md), pressing a dilemma against the computational theory of
phenomenal consciousness.

Reference:
- Maudlin, T. (1989). Computation and consciousness. *The Journal of Philosophy, 86*(8), 407–432.
