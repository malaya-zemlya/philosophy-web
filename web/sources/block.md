relevance (content): the paper in which Ned Block presents the *lookup-table* objection. A machine
producing conversation from a finite stored table (the "Aunt Bubbles" machine, the "Blockhead")
passes a behavioural test with "the intelligence of a toaster", so matching a mind's behaviour is
not sufficient for having a mind. This grounds the lookup-table node family in this web:
[lookup-table-mind](../concepts/lookup-table-mind.md), [lookup-table-possible](../claims/lookup-table-possible.md), [lookup-table-lacks-mind](../claims/lookup-table-lacks-mind.md),
and [lookup-table-objection](../arguments/lookup-table-objection.md). The target is the sufficiency of *behaviour*, a different
property from the sufficiency of *functional organisation* attacked by Block's Chinese Nation, which
carries its own source node ([block-1978](block-1978.md), "Troubles with Functionalism", 1978).

key passage:
- Block, Ned (1981), "Psychologism and Behaviorism," *The Philosophical Review* 90(1): 5–43. This is
  where the lookup-table / Aunt Bubbles machine appears.
