relevance (content): the canonical statement that an organism is conscious if there is something it
is *like* to be it, and that this subjective character of experience resists capture by any
objective, third-person description. The touchstone for [qualia](../concepts/qualia.md) and
[phenomenal-consciousness](../concepts/phenomenal-consciousness.md).

Reference:
- Nagel, T. (1974). What is it like to be a bat? *The Philosophical Review*, *83*(4), 435–450.
