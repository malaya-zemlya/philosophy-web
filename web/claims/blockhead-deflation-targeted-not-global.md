gloss: the conceptual-horn deflation ([imaginary-blockhead-no-secure-verdict](imaginary-blockhead-no-secure-verdict.md)) is *targeted*. It
withdraws warrant from the "no mind" verdict about the cosmic lookup table because that object is a
far extrapolation from any paradigm mind ([lookup-table-physically-unrealisable](lookup-table-physically-unrealisable.md)) — alien in both
scale and internal structure. It leaves untouched the intuition about a brain-scale, brain-structured
silicon isomorph, a near neighbour of paradigm minds. So horn 2 of [blockhead-dilemma](../arguments/blockhead-dilemma.md) does
not "prove too much" against [substrate-independence](substrate-independence.md).

scope: answers the over-generalisation worry about [blockhead-dilemma](../arguments/blockhead-dilemma.md). It does NOT assert the
isomorph intuition is *correct*, only that it is not impugned by the very move that impugns the
blockhead intuition. The asymmetry is epistemic (distance from exemplars), not a smuggled metaphysics
of which features constitute mind — so it does not beg the question against Block.

who would deny it: Block, by relocating his verdict from *scale* to *kind* — denying mind to brute
storage of any size, so that a scale-based asymmetry (he says) misses the point. That rejoinder is
taken up in the weakest-premise of [intuition-domain-asymmetry](../arguments/intuition-domain-asymmetry.md).

Distinct from [imaginary-blockhead-no-secure-verdict](imaginary-blockhead-no-secure-verdict.md): that is the deflation; this bounds its
reach.
