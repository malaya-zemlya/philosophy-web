gloss: a sufficiently reflective finite system can in principle be engineered to have complete,
transparent access to *all* of its own current state from within — so the limits on introspection are
contingent and removable, not structural. This is the optimist denied by
[inside-vantage-access-limited](inside-vantage-access-limited.md).

scope: the substantive reading is *internal* and *complete*. (It is trivially true if "access" is
allowed to be *external* instrumentation — but an external instrument is not part of the system whose
state is represented, so that reading abandons the self-access claim.)

status — why contested: attacked by [finite-self-model-incomplete](../arguments/finite-self-model-incomplete.md) on capacity grounds —
a proper part of a finite state cannot losslessly encode the whole.

who would deny it: anyone holding the finite self-model bound (the impossibility argument above).

Distinct from [llm-functional-introspection](llm-functional-introspection.md): that asserts *limited, unreliable* self-access
exists; this asserts *complete* self-access is achievable — the strong thesis the impossibility
argument targets.
