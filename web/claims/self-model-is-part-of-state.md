Any representation a finite system forms of its own current state is itself realised *within* that
same state. There is no free, state-external register to hold it; the vehicle of the self-model
competes for the same bounded resource as everything else the system is doing.

### Scope

This is the *internal* case, and it is the load-bearing premise of the capacity argument
[finite-self-model-incomplete](../arguments/finite-self-model-incomplete.md). Denying it is precisely the move to an *external* observer
or instrument, which is not part of the represented state and so faces no capacity bound — the
asymmetry behind [inside-vantage-access-limited](inside-vantage-access-limited.md).

### Who would deny it

A substance dualist, for whom the representing mind is not part of the physical state; or anyone
modelling self-access as done by external instrumentation rather than from within — which concedes the
point for the genuinely internal case.

### Distinct from neighbours

Distinct from [inside-vantage-access-limited](inside-vantage-access-limited.md): that is the *conclusion* about access poverty;
this is the structural *premise* (no state-external register) that, together with finiteness, forces
it.

### In plain terms

When a system makes a picture of "how I am right now," that picture has to be stored somewhere — and
the only place is in the system itself. So the picture is part of what it is trying to picture, and it
uses up some of the same limited space. (An outside camera doesn't have this problem, because it isn't
part of you — which is exactly why looking from outside can see more than looking from inside.)
