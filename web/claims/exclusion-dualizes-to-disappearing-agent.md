The [causal-exclusion argument](../arguments/causal-exclusion.md) (Kim, [2005](../sources/kim-2005.md)) makes
consciousness all sink and no source: incoming arrows only, since the physical base already does
every bit of causal work, threatening [epiphenomenalism](../positions/epiphenomenalism.md)
([irreducible-mental-causally-excluded](irreducible-mental-causally-excluded.md)). Reverse the arrows and the demand becomes that
the libertarian agent be all source and no sink — and the dualized exclusion argument is exactly
Pereboom's [disappearing-agent objection](../arguments/disappearing-agent.md) (Pereboom, [2014](../sources/pereboom-2014.md)), and the luck
objection generally: prior causes exclude the agent from originating action just as physical causes
exclude qualia from producing behavior.

In arrow terms the reversal is exact. The arrows are relations of causal settling among states
([what the arrows are](../concepts/consciousness-freewill-duality.md)): epiphenomenalism is the thesis
that the conscious state is a node with only *incoming* arrows, while libertarian origination
demands a node with only *outgoing* ones — two diagram-shapes that are literal reversals of each
other. The paired arguments are the mirrored squeezes on those shapes. Exclusion deletes the mental
node's outgoing arrows by showing each already duplicated by an arrow from the physical base;
the disappearing-agent objection deletes the agent's originating arrows by showing that everything
reaching the decision already arrives along arrows from prior events.

The claim asserts a structural identity of argument-shape, not that either argument succeeds. It is
one of the dictionary entries in which arrow-reversal lands on a position that already exists in
the other literature under its own name — the kind of independent rediscovery that serves as
evidence for the [mirror thesis](consciousness-freewill-mirror.md) in
[consciousness-freewill-duality](../arguments/consciousness-freewill-duality.md).

### In plain terms

One famous argument says the mind can't *cause* anything: the brain's physics already does all the
work, so experience is left as a passenger receiving input. Another famous argument, from a
different debate, says an agent can't *originate* anything: earlier causes already settle each
decision, so the "free chooser" vanishes from the story. This claim says these are the same
argument pointed in opposite directions — one squeezes out consciousness at the receiving end, the
other squeezes out the agent at the sending end.
