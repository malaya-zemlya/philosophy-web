# philosophy-web

A semantic web of philosophical objects that AI characters debate into, and that you later
write treatises *from*. Direct-writing design with safeguards (grep-before-create + a
propose-only reconciler + a deterministic integrity script).

## Layout
```
CLAUDE.md                     always-on project rules (lean; in context every session)
schemas/                      per-type node schemas — read schemas/<type>.md before
                              creating/editing a node; legal edges enforced by lint.py
web/                          the distilled graph (durable substrate)
  concepts/ claims/ arguments/ questions/ positions/ sources/ characters/
  INDEX.md                    GENERATED by lint.py — backlinks; do not hand-edit
transcripts/                  append-only debate logs + treatise drafts
proposals/                    merge proposals from the reconciler (await your approval)
scripts/lint.py               integrity check + backlink/index generator (no model calls)
scripts/pre-commit            git hook wrapper for lint.py
.claude/
  skills/web-protocol/        the node schema + write discipline (preloaded into debaters)
  agents/                     character subagents (mu-043, reconciler, ...)
  commands/                   /debate /claim /reconcile /treatise
```

## Mental model (read this once)
- **Two artifacts.** `transcripts/` = rhetoric, linear, append-only. `web/` = atomic,
  deduplicated, cross-referenced nodes. One utterance is not one node.
- **Characters split in two.** Stable voice/method = the subagent in `.claude/agents/`.
  Evolving commitments = the `web/characters/<id>.md` node, which the character re-reads
  each turn. The web is the character's memory between (stateless) turns.
- **Edges are one-directional, by id.** Backlinks are computed by `lint.py`, never stored.
- **Nothing merges silently.** Suspected duplicates → `proposals/` → your call.

## Setup
```bash
pip install pyyaml
ln -sf ../../scripts/pre-commit .git/hooks/pre-commit   # after git init
python scripts/lint.py                                   # sanity check + build INDEX.md
```

## Adding characters
1. Copy `.claude/agents/mu-043.md` → set name/voice/method (Rinat = Cartesian denier;
   Sophia = mediator; the communitarian/anthropocentric fourth).
2. Create `web/characters/<id>.md` from the mu-043 portfolio template.
That's it — `/debate <topic>` will invoke them in turns.

## Daily use
- `/debate <topic>` — run turns; characters write nodes, you hold turn order.
- `/claim <your idea>` — add your own node, your wording preserved (no AI rephrasing).
- `/reconcile` — surface duplicate candidates for review.
- `/treatise <thesis>` — draft a piece from the recorded web, attributed by `author`.

## Known limits
- Dedup is grep + discipline + a propose-only reconciler. When the graph grows past a few
  hundred claims and paraphrase-duplicates slip through, add embedding-based similarity
  (e.g. an MCP tool the reconciler calls). The architecture already isolates that change to
  the reconciler.
- The orchestrator/main session accumulates the transcript; long debates still grow its
  context. Start a fresh session per debate session; the web persists the state.
